Recursively resolves all nested promises in an object or array.

This function takes an input that can be a promise, an array, an object, or a primitive value, and recursively resolves any promises contained within. It supports nested structures and preserves the original structure of the input while resolving promises.

validateConfig
validateConfig(config: Record, envVariablesClass: ClassConstructor)

Validates and transforms a configuration object based on a class schema.

This function uses class-transformer to transform a plain object into an instance of a class and class-validator to validate the instance. If validation fails, it throws an error with detailed information about the validation errors.
